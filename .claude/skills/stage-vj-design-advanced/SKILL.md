# 前端 WebGL 舞台级高级 VJ 设计体系

## 核心痛点

普通 ThreeJS/Canvas VJ 容易陷入：大量方块、圆环、线条无规则堆在一起，动效速度统一、色彩杂乱、无空间层次，廉价电子风，撑不起商业演出 / 夜店舞台大屏。

## 一、高级感 VJ 4 条核心设计法则（前端写 Shader 时严格遵守）

### 1. 分层空间逻辑 —— 杜绝所有元素在同一平面堆叠

舞台视觉必须 3 层空间分离，每层运动逻辑完全不同，天然拉开高级景深，不会挤成一团：

#### 远景底层（慢运动层）
- 低对比度、弱纹理、极低运动速度（1/4BPM 缓慢流动）
- 星云、渐变云雾、弱噪波流体、远距离网格，不出现硬质几何体
- 作用：做画面基底，承载氛围感，全程不抢视觉重心

#### 中景主体层（节奏层）
- 少量精简几何（单一种类，不要方块 + 圆环 + 三角同时堆）
- 仅跟随鼓点 / BPM 做形变、缩放、位移，每首曲目只保留 1~2 种基础形体
- 禁止：一次性同时渲染十几种几何体叠加

#### 前景高光层（瞬时爆发层）
- 细线、粒子、光刃、色散辉光、扫描线；仅在 Drop、重鼓点瞬间闪现，平时低透明度隐藏
- 特点：出现时间短、消失快，作为画面点睛，不长期占用画面

**硬性约束**：同一帧画面，硬质几何体种类≤2 种，其余全部用流体 / 噪波 / 渐变软纹理填充。

### 2. 色彩系统：低饱和基底 + 局部高对比高光，拒绝全画面高饱和

舞台大屏 LED 高亮度，高饱和纯色堆砌会刺眼、发灰、廉价：
- 整场固定一套有限色板（主色 1 个、辅色 2 个、高光霓虹色 1 个），全程不新增杂色
- 基底统一低明度低饱和（灰蓝、炭黑、暗紫、哑光青）
- 高光仅用于前景线条 / 粒子，局部提亮，不铺满画面
- Shader 内置色彩过渡插值，BPM 段落自动渐变色调，无生硬跳色
- 禁用大面积纯白、纯荧光绿、艳红铺满画布

### 3. 运动节奏差异化（解决所有物体同步动的塑料感）

廉价 VJ 通病：所有几何体缩放、旋转速度一模一样，画面呆板杂乱。前端通过时间偏移分层控速：
- 底层流体：time * 0.1 超慢速流动
- 中层几何形变：time * 0.4 跟随小节缓慢变形
- 前景闪光粒子：beat 脉冲触发瞬时爆发（仅 0.2s）
- 旋转、位移、波纹三个运动维度错开速度，不同层不联动
- 加微小随机时间偏移，同类型几何体运动不同步，弱化复制堆叠感

### 4. 光学后期统一质感 —— 高级感核心（前端必备后处理）

纯几何体裸渲染会像 demo 练习，必须叠加 WebGL 后处理管线：
- 全局柔光模糊（少量），柔化硬边几何棱角
- 轻微色差色散（RGB 通道偏移），模拟大屏光学质感
- 胶片颗粒噪波叠加，抵消数字画面冰冷感
- 边缘暗角 vignette，视觉自动向画面中心收拢
- 高光 bloom 辉光，仅高亮元素发光，几何体本体不发光

## 二、前端技术落地方案（ThreeJS + Shader / PixiJS）

### 1. 素材 / 图形选型：少硬几何，多程序化软纹理

减少立方体、圆环、三角等硬几何体使用比例，70% 画面用程序化流体噪波：
- 核心 Shader：Simplex 噪声、fbm 分形噪波、voronoi 沃罗诺伊细胞纹理、流体置换
- 几何仅做视觉锚点：比如整场只用单一圆环阵列，不叠加方块、五角星、网格
- 阵列做疏密控制：几何体只在画面局部聚集，其余区域留白，拒绝全屏铺满阵列

### 2. 规避堆叠廉价感的编码设计规范

- **几何体阵列做衰减遮罩**：用圆形 / 渐变遮罩，画布四周几何体透明度递减，中心集中，四周留白，不会满屏密密麻麻几何体
- **阵列尺寸差异化**：同一阵列几何体设置随机基础大小，不要统一尺寸复制，弱化复制感
- **限制渲染数量**：中型演出画面几何体总数不超过 300 个，大量元素改用粒子贴图而非实体 Mesh
- **动态消隐机制**：平缓段落自动降低几何体数量，高潮再缓慢增加，全程控制画面密度

### 3. 演出交互逻辑（适配舞台 VJ 实时操控）

前端必须提供可控参数面板，对应现场 DJ 节拍：
- BPM 同步时间控制器，音乐节拍传入 Shader time
- 4 组可一键切换视觉预设（前奏舒缓、主歌、Drop 高潮、尾奏消散）
- 实时可调参数：bloom 强度、噪波流动速度、几何密度、高光强度、色调偏移
- MIDI 控制器对接，旋钮绑定画面形变、色彩、粒子爆发

## 三、快速实操小技巧（写代码立刻见效）

- 删掉一半几何体，用 fbm 噪波背景填充空白区域
- 所有硬质图形加柔光模糊后处理，弱化尖锐棱角
- 同一画面只保留一种基础几何形态
- 高光辉光只给到线条粒子，主体几何体关闭自发光
- 增加暗角，强制视觉中心收敛，避免画面散乱

## 五、自查清单（调用此 skill 后必做）

- [ ] 三层空间分离：远景慢运动、中景单一种类几何、前景瞬时高光
- [ ] 硬质几何体种类 ≤ 2 种
- [ ] 色彩系统：低饱和基底 + 局部高对比高光，固定 2-3 主色
- [ ] 运动差异化：各层时间系数不同（0.1x, 0.4x, beat 触发）
- [ ] 后处理管线：bloom + 暗角 + 轻微色散/颗粒
- [ ] 70% 画面为程序化软纹理（fbm/simplex/voronoi）
- [ ] 阵列有衰减遮罩、尺寸差异、数量限制
